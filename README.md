# WordPress
This project contains scripts for deploying Wordpress for Windows and linux.

(Recommend PHP 7.4 or greater and MySQL 5.7 or MariaDB version 10.3 or greater).
## for Windows
**_OS: Windows Server 2019_**

**_Software:_**
- PHP 7.4.33
- MySQL 5.7.40 
- Wordpress 6.1.1
## for Linux
**_OS: Debian 11_**

**_Software:_**
- Apache2 2.4.54
- MariaDB 10.5.18
- PHP 7.4.33
- Wordpress 6.1.1
